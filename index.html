Question Answering platform with the help of Natural Language

As part of my NLP project, I worked on the Stanford Question Answering Dataset (SQuAD), a large reading comprehension dataset consisting of over 100,000 question-answer pairs on 500+ Wikipedia articles. The goal of the project was to find the text for any new question and context provided, where the answer to a question is always a part of the context and a continuous span of context. To achieve this, I first implemented an unsupervised learning model using Euclidean distance to detect the sentence with the minimum distance from the question. However, the accuracy of this model was only 45/100, so I switched to cosine similarity which improved the accuracy to 63/100. I attribute this improvement to the fact that cosine similarity takes into account the alignment and angle between vectors, which is important for vectorial representations. Despite the success of the unsupervised learning model, I realized that it did not leverage the rich data with target labels that we were provided with. Nonetheless, I am proud of the results I achieved, particularly given the simplicity of the approach, and credit the Facebook sentence embedding for contributing to the decent performance.
